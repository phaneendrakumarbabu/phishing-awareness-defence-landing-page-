Process

Three steps to
total phishing immunity

PhishGuard AI works silently in the background, analysing every page you visit and alerting you before harm is done.

// STEP 01
🔍

Real-Time URL Analysis

Every URL you visit is instantly scanned. Our ML model extracts over 30 features including domain age, HTTPS validity, URL structure, and lexical patterns.

// STEP 02
🧠

ML Classification Engine

A trained Random Forest / neural classifier processes extracted features and outputs a phishing probability score in milliseconds — no cloud latency.

// STEP 03
🛡️

Alert & Educate

Suspicious pages trigger a clear, actionable warning. Users learn why the site was flagged — building cyber awareness, not just blocking silently.

Why It Matters

Phishing is the #1 cyber threat vector

Traditional blocklists are reactive — they only catch known threats. Our proactive ML model detects never-before-seen phishing pages by understanding the anatomy of deception, not just a list of known bad URLs.

  • Works on zero-day phishing domains
  • No reliance on slow, stale blocklists
  • Lightweight — runs fully in-browser
  • Privacy-first — no URL data sent to servers

Under the Hood

ML-powered,
not blocklist-dependent

The core engine extracts lexical, host-based, and content-based features from every URL, running them through a trained classification model deployed directly in the extension.

  • Random Forest / Gradient Boosting classifier
  • 30+ hand-engineered URL features
  • Trained on curated phishing + legitimate datasets
  • Continuous model improvement pipeline
  • Python (scikit-learn) model exported to JS runtime
  • Chrome Extension Manifest V3 compatible
